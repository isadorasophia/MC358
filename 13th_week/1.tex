\problem{1.}
  Prove que os grafos $n$-cubos, $ Q_n $, possuem um circuito euleriano, para todo $ n \geq 2 $ para $ n $ par. 

\solution
  Um grafo $n$-cubo possui cada um dos vértices conectados a uma cadeia $n$ de bits. Ou seja, se $Q_n$, com $n \geq 2$ par, cada vértice está conectado a $n$ vértices (ie. possui grau par).

  Pelo teorema de Euler, um multigrafo conexo é Euleriano se $g(v)$ é par $\forall v \in V$. Como o grau de todos os vértices é par, como apresentado anteriormente, $Q_n$ possui um circuito euleriano. \GG
