\problem{4.}
  \textbf{Prove} que uma função do tipo $ f(x) = mx + b $, com $ m \neq 0 $ é \textbf{bijetora}.

\solution
    Se uma função é bijetora, isto é, ela deve ser \textbf{injetora} e \textbf{sobrejetora}.

  \part
    Assim, a função $ f(x) = mx + b $ é injetora se $f(a) \not = f(a') \forall a \not = a'$. Basta considerar que para valores arbitrários da função tal que $f(x) \not = f(x')$, temos $mx + b \not = mx' + b \rightarrow x \not = x'$, uma vez que $m \not = 0$. Prova-se que a função $f(x)$ é de fato injetora.

  \part
    Finalmente, a função é sobrejetora se em $f: A \rightarrow B$, $\forall b \in B \exists a \in A$ t.q. $f(a) = b$. Ou seja, para todo $x'$ deve existir $x$ tal que $f(x) = x'$. Basta substituir pela função em estudo, $mx + b = x' \rightarrow \cfrac{x' - b}{m} = x$, com $m \not = 0$. Prova-se que a função $f(x)$ é sobrejetora e, portanto, bijetora.
