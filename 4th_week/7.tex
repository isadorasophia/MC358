\problem{7.}
  Prove que, se $ p $ é um inteiro ímpar, então a equação $ x^2 + x - p = 0 $ não tem solução inteira.

\solution
  Para que a equação tenha uma solução inteira, $x^2 + x = p$.

  \part
  Caso $x$ seja par, $\exists k \in \mathbb{Z}$ t. q. $n = 2k$. Ou seja, $x^2 + x = 4k^2 + 2k = 2(2k^2 + k)$, chamando $k' = 2k^2 + k$, temos que $x^2 + x = 2k'$, ou seja, o resultado será sempre par.

  \part
  Caso $x$ seja ímpar, $\exists k \in \mathbb{Z}$ t. q. $n = 2k + 1$. Ou seja, $x^2 + x = 4k^2 + 4k + 1 + 2k + 1 = 4k^2 + 6k + 2 = 2(2k^2 + 3k + 1)$, chamando $k' = 2k^2 + 3k + 1$, temos que $x^2 + x = 2k'$, ou seja, o resultado será sempre par - independente de que $x$ seja ímpar ou par!

  Assim, caso $x$ seja ímpar, $x^2 + x = p$ não produz uma solução inteira, uma vez que $p$ deve ser par.
