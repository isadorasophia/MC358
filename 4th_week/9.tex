\problem{9.}
\textbf{Prove ou disprove}: se $ n $ é um número natural, então $ 2^{2^n} + 1 $ é primo. (Dica: Escreva (e execute) um programa (simples) que procura um contra-exemplo de modo que você possa apresentar sua resposta na forma: $ 2^{2^n} = m = k * l $ explicitando $ n, m, k, l $. Não precisa incluir o seu programa como parte de sua resposta.)

\solution
    A partir de um programa simples feito na linguagem C, foi possível encontrar os valores necessários para contrariar a proposição. Assim, o caso encontrado é em que 
    \begin{equation}
    n = 5, m = 4294967297, k = 641, l = 6700417
    \end{equation}
