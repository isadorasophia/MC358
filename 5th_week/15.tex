\problem{15.}
  A função piso leva cada número real $ x $ no maior inteiro que é menor ou igual a $ x $. Ela é denotada por $ \lfloor x \rfloor $.

  A função teto leva cada número real $ x $ no menor inteiro que é maior ou igual a $ x $. Ela é denotada por $ \lceil x \rceil $.

  \textbf{Prove} que para todo $ x $ real e todo $ n $ natural valem $ \lfloor x + n \rfloor = \lfloor x \rfloor + n $ e $ \lceil x + n \rceil = \lceil x \rceil + n $.

\solution
   \part
   Sendo $x$ real e $n$ natural, $ \lfloor x + n \rfloor = \lfloor x \rfloor + n $ e considere o inteiro $p$ t.q. $p \leq x \leq p + 1$.

   \part
   Em $\lfloor x + n \rfloor$, $p + n \leq x + n \leq p + n + 1$. Então $\lfloor x + n \rfloor$ é $p + n$, que equivale a $\lfloor x \rfloor + n $, pois $p = \lfloor x \rfloor$.

   \part
   Analogamente, em $\lceil x + n \rceil$, $p + n \leq x + n \leq p + n + 1$. Então $\lceil x + n \rceil$ é $p + n + 1 $, que equivale a $\lceil x \rceil + n$, pois $p + 1 = \lceil x \rceil$.
