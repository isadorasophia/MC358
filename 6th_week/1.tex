\problem{1.}
Seja $ A $ um conjunto infinito. Prove que se existir uma sequência na qual cada elemento de $ A $ aparece pelo menos uma vez, então $ A $ é contável.

\solution
    Considere a sequência infinita $x_1, x_2, ...$ em que cada elemento de $ A $ aparece, pelo menos, uma vez. Defina $f: A \rightarrow \mathbb{Z^+}$ em $f(a)$ é o menor índice $i$ tal que $a = x_i$.
Como cada elemento de $ A $ aparece pelo menos uma vez na sequência, esse índice sempre existe. 

    Definindo $B = f(A)$ como a imagem de $f$. Logo, $B \subseteq \mathbb{Z^+}$ e $B$ é contável.

    Então, em $f: A \rightarrow B$, a função é bijetora: pois é sobrejetora, todo elemento $b \in B$ é gerado a partir de um $f(a)$ pela própria definição de imagem, e injetora, $f(a) \not = f(a') \forall a \not = a'$. Ou seja, $|A| = |B|$ e, portanto, $A$ é contável. \CQD
