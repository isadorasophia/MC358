\problem{3.}
Dada uma coleção de $ n $ conjuntos $ A_1, A_2, \dots, A_n $, prove que $ (A_1 \cup A_2 \cup \dots \cup A_n)^c = A_1^c \cap A_2^c \cap \dots \cap A_n^c $, onde $ A^c $ denota o complemento de $ A $.

\solution
    Prova por indução em $n$: \\

    \textbf{Base:} Se $n = 1$, $(A_1)^c = A_1^c$. \CQD

    \textbf{Hipótese:} Assuma que $\left(\bigcup\limits_{i = 1}^n A_i\right)^c = \bigcap\limits_{i = 1}^n A_i^c$.

    \textbf{Passo:} Suponha $\left(\bigcup\limits_{i = 1}^{n +1} A_i\right)^c = \left(\bigcup\limits_{i = 1}^{n} A_i \cup T \right)^c$, com $T = A_{n + 1}$.

    P.H.I. e pela lei de De Morgan, temos que $\left(\bigcup\limits_{i = 1}^{n} A_i \cup T \right)^c = \left(\bigcup\limits_{i = 1}^{n} A_i\right)^c \cap (T)^c = \bigcap\limits_{i = 1}^{n} A_i^c \cap T^c$. Como $T^c = A_{n + 1}$, temos que $\left(\bigcup\limits_{i = 1}^{n +1} A_i\right)^c = \left(\bigcap\limits_{i = 1}^{n + 1} A_i \right)^c$. \CQD
