\problem{2.}
    Use indução forte para mostrar que todo inteiro positivo $ n $ pode ser escrito como a soma de diferentes potências de 2. Dica: no passo indutivo trate separadamente os casos em que $ n + 1 $ é par, e em que $ n + 1 $ é ímpar.

\solution
    Usando indução forte em $n$:

    \textbf{Base:} Para um inteiro positivo $n = 1$, é possível escrever $n = 2^0$. \CQD

    \textbf{Hipótese:} Todo inteiro $n > m \geq 1$ pode ser escrito como a soma de diferentes potências de 2.

    \textbf{Passo indutivo:} Caso $ n + 1 = 2m $ par, então P.H.I. basta tomar $m = \sum_{k = 0} 2^{p_k}$. Ou seja, $n + 1 = 2m = 2 \sum_{k = 0} 2^{p_k} = \sum_{k = 0} 2^{p_k + 1}$.

    Caso $ n + 1 = 2m + 1 $ ímpar, então P.H.I. basta tomar $n + 1 = \sum_{k = 0} 2^{p_k + 1} + 2^0$, uma vez que $0 \not \in {\{p_1, p_2, ..., p_k, p_{k + 1}\}}$, pois o número é par. \CQD
